### Museum / Carol visits Ross in the museum

```
Ross : Hi.

Carol : Is this a bad time?

Ross : No, it's...it's... the Stone Age. You look great. I uh... I'm not able to stand.

Carol : Oh, sorry... Thanks. You look good too.

Ross : Ah, well, in here... So What's new? Still, uh...

Carol : A lesbian? Yeah..

Ross : Well... how's the family? Why are you here?

Carol : I'm pregnant.

Ross: Pregnant... pregnant?
```

#### Vocabulary & Idiom
```
- Is this a bad time? 시간 있어?
    - Do you have time?
    - Are you free?
    - Do you have the time (주의: 지금 몇시죠?)
- in here... 여기서는
- what's new? 요즘 어때?
- stone age 석기시대
```

#### Pronunciation
```
- Is this a bad(d는 생략) time?
- I'm pregnant (t는 생략)
```

#### Pattern
```
I'm (now) able to 나는 (이제) ~할 수 있어
- I'm able to drive. 운전할 수 있어
- I'm able to pick you up. 데리러 갈 수 있어
- I'm able to start working. 일 시작할 수 있어
- I'm able to apply for the job. 그 일에 지원할 수 있어
- I'm able to pay you back. 너한테 돈 갚을 수 있어.
```
